# Aura Travel User Manual

## Plan A Trip

1. Open **Experience Weaver**.
2. Describe a trip in natural language, such as `peaceful beach trip with cultural food for 3 days in Goa`.
3. Review the generated itinerary, accommodation, restaurant suggestions, and route map.
4. Select **Start Trip** to move the itinerary into Live Trip mode.

## Explore

Use **Explore** to browse curated activities by destination, category, search text, and map view.

## Live Trip

Live Trip tracks the current activity and lets you simulate common travel changes:

- Rain
- Heat
- Overcrowding
- Low energy
- Venue closure

When a conflict is detected, Aura Travel suggests alternatives that preserve the original trip intent.

## Bookings

Bookings is a mock flow for demo purposes:

1. Search a flight and hotel.
2. Select options.
3. Enter contact details.
4. Confirm the mock payment.

No real booking or payment is performed.

## Account

Create an account or sign in to save profile preferences. Admin users can open the dashboard to manage activities.

## Admin

Seeded admin users can:

- View catalog counts and analytics.
- Create activities.
- Edit activity details.
- Delete activities.
